# Sistema de Gestão de Costura ✂️

##  Objetivo
Sistema online para gestão de costura, com foco em:
- Cadastro de clientes
- Registro de serviços
- Organização dos trabalhos
- Estrutura para login, estoque e vendas futuras

## Arquitetura Utilizada 
- Back-end (até o momento)
•	Java 25
•	Spring Boot
•	Spring Web
•	Spring Data JPA
•	*Hibernate 
•	API REST

- Banco de Dados: PostgreSQL
